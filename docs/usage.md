# 利用手順

## 1. 初回起動
1. アプリを起動すると SQLite データベースが自動生成されます。
2. まずは「型番」「NG理由」「処置内容」のマスタを登録してください。

## 2. NG 案件の登録
1. 「新規登録」から案件を作成します。
2. 初回検査結果と NG 理由・処置内容を入力します。
3. 登録後、案件詳細画面で履歴確認・再検査追加ができます。

## 3. 追加操作
- 再検査を追加する場合は「再検査登録」を使います。
- 備考は「備考更新」から変更できます。
- CSV 出力やバックアップはメインフォームから実行できます。

## 4. テスト実行
```powershell
dotnet test NgProductManager.sln
```
